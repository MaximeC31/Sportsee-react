# SportSee

Tableau de bord d'analytics sportif avec React, intégrant des graphiques interactifs et des appels API pour afficher des données d'utilisateur.

## Comment le démarrer

Ce projet est une application React avec un backend API. Pour le lancer :

1. **Prérequis** :

    - Node.js (version 12.18 ou supérieure)
    - npm ou yarn

2. **Installation** :

    - Clonez le dépôt
    - Installez les dépendances du frontend : `npm install`
    - Installez les dépendances du backend : `cd backend && npm install`

3. **Lancement** :
    - Démarrez le backend API : `cd backend && npm run dev` (le serveur tourne sur http://localhost:3000)
    - Démarrez le frontend : `npm run dev` (l'application s'ouvre sur http://localhost:5173)
