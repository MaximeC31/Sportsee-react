# SportSee Backend

Backend API pour le tableau de bord d'analytics sportif SportSee, fournissant les endpoints nécessaires pour récupérer les données utilisateur, l'activité, les sessions moyennes et les performances.

## Comment le démarrer

Ce backend est une API Node.js/Express. Pour le lancer :

1. **Prérequis** :

    - Node.js (version 12.18 ou supérieure)
    - npm ou yarn

2. **Installation** :

    - Assurez-vous d'être dans le dossier `backend`
    - Installez les dépendances : `npm install` ou `yarn`

3. **Lancement** :
    - Démarrez le serveur en mode développement : `npm run dev` ou `yarn dev`
    - Le serveur tourne sur http://localhost:3000

**Note :** Seuls les utilisateurs avec les ID 12 et 18 sont actuellement mockés.
